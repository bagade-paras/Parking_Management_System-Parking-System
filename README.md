# Parking_Management_System-Parking-System

Parking Management System

🚗 **Smart Parking System**

A robust, full-stack PHP & MySQL web application designed to streamline urban parking management. The platform bridges the gap between drivers and parking spaces, allowing users to seamlessly book visual parking slots ahead of time and pay via digital methods (UPI/Razorpay), while granting admins granular control over locations, dynamic slot availability, and real-time revenue analytics.

**Backend:** PHP, MySQL (Relational Database)

**Libraries:** Chart.js (Analytics), PHPMailer (Email Notifications), Razorpay API (Payment Gateways)

---

## ✨ Key Features

### 👤 User End-to-End Flow

* Interactive Visual Booking: A step-by-step 4-stage wizard allowing users to choose a location (with Google Maps previews), set exact durations, and visually select available slots from a dynamic grid (Green = Available / Red = Booked).

* Smart Dashboard: Features dynamic live countdown timers for active sessions, automated email reminders, booking extension capabilities, and PDF receipt downloads.

* Vehicle & Profile Management: Easily link multiple vehicles (Cars/Bikes) to a single user profile.

### 👑 Admin Control Panel

* Dual-Role Administration: Super Admins have macro visibility, with the ability to provision localized Sub-Admins restricted to managing specific properties.

* Inventory Control: Full CRUD operations on geographic locations and individual spot parameters (Available, Booked, Maintenance).

* Data-Driven Analytics: Visualized tracking metrics via Chart.js detailing occupancy velocity, historical booking trends, and daily/weekly/monthly revenue tracking.
